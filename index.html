Project Overview:

The goal of this project is to create a chatbot that can understand and respond to user input based on intents. The chatbot is built using Natural Language Processing (NLP) library and Logistic Regression, to extract the intents and entities from user input.

Dataset:

The dataset used in this project is a collection of labelled intents and entities. The data is stored in a list.

  • Intents: The intent of the user input (e.g. "greeting", "budget", "about")
  • Entities: The entities extracted from user input
  • Text: The user input text
